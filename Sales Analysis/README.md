# 📊 Sales Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing sales data to understand business performance, customer behavior, and product trends. The dashboard provides key insights using interactive visualizations.

---

## 🎯 Objective
- Analyze total sales and profit performance  
- Identify top-performing products and categories  
- Understand regional sales distribution  
- Evaluate customer purchasing behavior  
- Improve business decision-making  

---

## 🛠 Tools & Technologies
- Power BI (Dashboard & Visualization)
- SQL (Data Analysis)
- Python (Data Cleaning)

---

## 📂 Dataset Description
The dataset includes:
- Order ID
- Product Category
- Sales Amount
- Profit
- Quantity Sold
- State / Region
- Customer Details

---

## 📊 Dashboard Features
- KPI Cards:
  - Total Sales Amount  
  - Total Profit  
  - Total Quantity  
  - Average Order Value  

- Charts:
  - Sales by State  
  - Quantity by Category  
  - Profit Analysis  
  - Time-based Trends  

---

## 📈 Key Insights
- Total Sales: ₹438K  
- Profit: ₹37K (~8.4%)  
- Clothing category contributes highest sales  
- Maharashtra generates highest revenue  
- Furniture category has lower performance  

👉 Sales dashboards help identify top products, customer trends, and growth opportunities. 1  

---

## 🧠 Business Insights
- Focus on high-performing categories increases revenue  
- Regional targeting improves marketing efficiency  
- Low-performing categories need improvement strategy  

👉 Sales analysis helps identify top products, customer behavior, and future opportunities. 2  

---

## 🚀 Conclusion
This project demonstrates how data analytics can be used to track performance, identify trends, and support data-driven decisions.
