# HR Analytics Dashboard

## Overview
This project analyzes employee attrition, workforce demographics, salary distribution, and department-wise performance using the IBM HR Analytics Dataset.

## Tools Used
- Power BI
- SQL
- Python
- Excel
- Jupyter Notebook

## Key Insights
- Employee Attrition Analysis
- Department-wise Distribution
- Age Group Analysis
- Salary Insights
- Workforce Demographics
