# Online Bookstore Sales Analysis

## Problem Statement
Analyze bookstore data to understand sales, revenue, and stock status.

## Dataset
- Books.csv
- Orders.csv
- Customers.csv

## Tools Used
- PostgreSQL
- SQL

## Steps Performed
1. Imported CSV files
2. Cleaned data
3. Used JOIN, GROUP BY
4. Found revenue and best-selling books

## Key Insights
- Fiction genre highest revenue
- Top 5 books generate most sales

## Conclusion
SQL helped derive useful business insights.
